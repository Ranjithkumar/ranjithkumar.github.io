**Problem:** How to run multiple thinking_sphinx instance in the single app itself?

**Solution:** this can usually be done by adding some settings to a file named sphinx.yml in your config directory.<!--more-->

**Example:**


    
    
    
    development:
        port: 3312
    
    test:
        port: 3313
    
    production:
        port: 3315
